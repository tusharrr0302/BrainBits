# 🧠 BrainBits

BrainBits is an intelligent developer productivity platform designed to help programmers balance **Data Structures & Algorithms (DSA)** practice and **Development** learning simultaneously.  
It integrates data from platforms like **LeetCode** and **GitHub** to track progress, generate AI-driven study schedules, and provide a real-time competitive coding environment.

The goal of **BrainBits** is to help developers avoid focusing on only one skill (DSA or Development) and instead grow **both simultaneously with structured guidance and analytics**.

---

## 📊 Use Cases

- Track coding progress
- Balance DSA and development learning
- Practice coding with an integrated editor
- Get AI-guided productivity insights
- Participate in real-time coding competitions

---

## 🚀 Features

### 1️⃣ Progress Tracking Dashboard
BrainBits tracks a user's coding activity and development progress using data from:
- **LeetCode** for DSA practice
- **GitHub** for development contributions

The platform visualizes this data through:
- 📊 **Pie Charts**
- 📈 **Graphs**

These visual analytics help users clearly analyze their learning progress and maintain a balanced focus between **DSA and Development**.


### 2️⃣ AI-Powered Smart Schedule
BrainBits includes an **AI scheduling system** that helps users maintain consistency.

- Analyzes the **previous 2 weeks of activity**.
- Evaluates how well the user followed their schedule.
- Generates the **next personalized schedule**.

This ensures that users improve both **problem-solving skills and development skills simultaneously**.


### 3️⃣ Built-in Practice Environment
BrainBits provides an integrated coding environment so users can practice directly on the platform. Users can write, test, and practice code without leaving the platform. Features include:
- **Monaco Code Editor**
- Multiple language support:
  - HTML
  - CSS
  - JavaScript
  - C
  - C++
  - Java
  - Python


### 4️⃣ AI Mentor Tips
BrainBits acts like a virtual mentor by analyzing user performance and schedule adherence. It provides:
- Personalized **AI suggestions**
- Tips to improve productivity
- Guidance on maintaining a balanced learning routine
